Frequently Asked Questions (FAQs) for Bare Metal Servers +

  • 01. Can I reboot the bare metal server remotely by myself?

    Yes. Our Bare Metal Servers come with the Intelligent Platform Management Interface (IPMI). You can reboot the server and access the BIOS remotely.

  • 02. Can you provide the customized bare metal server based on our requirements?

    No. We do not provide customized bare metal servers. But you can order any additional service for your existing server, such as SSD/SATA disk, RAM, IPv4, etc.

  • 03. Can I install any software on your bare metal server?

    Yes. You can install any legal software on your bare metal server by yourself or by us. You will have total control of your bare metal server.

  • 04. Do you provide any control panels for the bare metal server?

    Yes. We can install SolidCP control panel for Windows bare metal server for free. SolidCP is a 100% open-source (and free) hosting control panel for Windows servers. You can use it to create IIS websites, MSSQL databases, etc. We can also install ISPConfig control panel for the Linux bare metal server for free. ISPConfig is a widely used open-source free Hosting Control Panel for Linux. You can use it to create websites, databases and email accounts etc.

  • 05. Can I use my own Windows server license on your bare metal server?

    No. Customer’s Windows Server License is not allowed. If you choose Windows Operation System, our technicians will activate Window with our license.

  • 06. Do you provide Graphic User Interface (GUI) for your Linux bare metal server?

    Our Linux bare metal server does not come with the GUI by default. But we can install it for you and enable the VNC access if you submit a support request. Or you can install the GUI by yourself through SSH.

  • 07. How can I access your Linux bare metal server?

    SSH is enabled for all the Linux bare metal servers. You will get the server IP, username and password once your server is ready. You can log into the bare metal server with the root permission via SSH.

  • 08. Can I upgrade/downgrade my bare metal server?

    Yes. You can upgrade/downgrade existing server to any available server plan. A new server will be prepared and data must be migrated to the new server by your-self.

  • 09. Can I buy additional IPs for the bare metal server?

    One bare metal server comes with 2 dedicated IPs by default (one primary IP and one IPMI IP). You can order up to 16 additional dedicated IPs per ARIN justification. We need the IP usage for each IP that you order from us. Price for additional dedicated IP is $2.00/month/IPv4.

  • 10. What security measures do you take to protect my bare metal server?

    Security is our top priority. We provide basic security protection. All servers are behind our hardware firewall. We blocked ports like 135-139, 445 and others in our network for security reason. We also monitor our network very closely and will stop attacks that come. But all clients must take responsibility of server security and data integrality.

  • 11. What is the meaning of “100Mbps Unmetered Bandwidth”?

    We do not measure or monitor the bandwidth usage for the individual bare metal server, so you don't have to worry about overage charges. But the maximum speed you can get is 100Mbps. “Unmetered” does not mean “Unlimited.” We do monitor our network very closely. If you use too much bandwidth and affect other client’s usage, we will ask you to reduce the bandwidth usage.

  • 12. What is the difference between Bare Metal Server and regular SSD dedicated server?

    Unlike regular SSD dedicated servers, bare metal servers do not come with RAID redundancy. Therefore, bare metal servers are mainly for test and personal use. For data sensitive business and product end services, regular dedicated servers are highly recommended. If you need further clarification, please contact us.
