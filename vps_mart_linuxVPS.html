Frequently Asked Questions (FAQs) for Linux VPS +

  • 01. What operating systems do you offer for your Linux VPS?

    You can choose CentOS 6.x-7.x 64-bit, Ubuntu Server 16.x-18.x 64-bit, Debian 7.x-9.x 64-bit, or OpenSUSE leap 42 for your Linux VPS. If you need other operating systems, please provide ISO image.

  • 02. Do you provide any control panels for your Linux VPS?

    Yes. We can install ISPConfig control panel for Linux VPS servers for free. ISPConfig is a widely used open-source free hosting control panel for Linux. You can use it to create websites, databases, email accounts, etc.

  • 03. Do you provide cPanel and WHM for your Linux VPS?

    We do not offer cPanel and WHM license. We can help you to install cPanel on your Linux VPS, but you need to order the cPanel license by yourself. cPanel provides 15 days of free trial and it will not work if you do not activate the license after 15 days.

  • 04. Do you provide Graphic User Interface (GUI) for your Linux VPS?

    Our Linux VPS does not come with GUI by default, but we can install it for you and enable the VNC access if you submit a support request. You can also install GUI by yourself through SSH.

  • 05. How can I access your Linux VPS server?

    SSH is enabled for all Linux VPS servers. You will get the server IP, username and password once your VPS is ready. You can log into the VPS server with the root permission via SSH.

  • 06. What is the virtualization platform for your Linux VPS?

    We use Microsoft Hyper-V virtualization platform for all our Linux VPS servers.

  • 07. What is the CPU processor of your Linux VPS host server?

    Different VPS host servers may come with different CPUs, but most of them are Dual Intel X5560 (2.8GHz) or Intel Eight Core Xeon E5-2670 (2.60 GHz) or E5-2680 (2.70 GHz) or E5-2690 (2.90 GHz).

  • 08. What is the meaning of “100Mbps Unmetered Bandwidth”?

    We do not measure or monitor the bandwidth usage for the individual Linux VPS, so you don't need to worry about overage charges. The maximum speed you can get is 100Mbps. “Unmetered” does not mean “Unlimited.” You share the Internet connections with other clients in the same host server. We do monitor our network very closely. If you use too much bandwidth and affect other client’s usage, we will have to ask you to reduce the bandwidth usage.

  • 09. What security measures do you take to protect my Linux VPS?

    Security is our top priority. All the physical servers are behind our hardware firewall. We blocked ports like135-139 and others in our network for security reason. We update our host server operating system regularly to install all security patches. We have DDoS protection in our network as well. We back up our host servers regularly and can restore your VPS from the backup with a small fee if there is anything happens to your VPS (like hacking). We also monitor our network very closely and will stop attacks that come.

  • 10. Can I install any software on my Linux VPS?

    Yes. You can install any legal software on your Linux VPS server by yourself or by us. You have total control of your Linux VPS server.

  • 11. Are resources shared or dedicated for your Linux VPS?

    CPU and Memory are dedicated. The bandwidth is shared with other clients in the same host server and rack.

  • 12. Can I reboot my Linux VPS server by myself?

    Yes. You can reboot the Linux VPS server and monitor the server status in our SolidCP control panel.

  • 13. Is it easy to upgrade my Linux VPS plan to a higher plan?

    Yes. It is free and only has 5 minutes of downtime. Please submit a sales ticket if you want to do so.

  • 14. Can I downgrade my Linux VPS plan to a lower plan?

    Yes. However, there will be 2 - 4 hours of downtime. There is a $25 one-time fee for the downgrade because of the effort required to resize the image .vhdx file.

  • 15. Can I buy additional dedicated IPs for my Linux VPS?

    Any Linux VPS server comes with one dedicated IP for free. You can order up to 8 additional IPs per ARIN justification. We need the IP usage for each IP that you order from us. Price for additional IP is $2.00/month/IP.

  • 16. Can you migrate my Linux VPS from other providers?

    We use Microsoft Hyper-V virtualization platform, so we can migrate your Linux VPS for free if you can provide us with the .vhdx or .vhd image file. If you do not have the image files, we can help you to set up your web sites and databases from your old providers for free too.
