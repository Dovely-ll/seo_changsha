Tips for VPS Usage

To reduce the risk of VPS slowness or freeze, please take the follow measure:

When you finish your work on the remote server, please quit RDP session through Start -> Account -> Sign Out, instead of closing the RDP window. That will clear the programs running in the background and release system resource. Or you can restart the server periodically.

Note: If you need your program running, please do not sign out nor restart.

To secure your server, please take necessary actions as follows:

  • Turn on Windows Firewall and block unnecessary incoming ports. See more details
  • Restrict remote access to RDP, database, etc.
  • Install antivirus software, such as Windows Defender, and make a full scan periodically. See more details
  • Install latest Windows Updates.
  • Set complex password for any service account. See more details
  • Back up any critical data to remote storage.

With TeamViewer, hassle-free remote desktop access is just a few clicks away. The following steps are included in the TeamViewer website. More details...

    2. Name your computer and set a password.

    3. Install TeamViewer on the computer you want to use to access the remote computer. Add the device to your partner list.

    4. To access the remote computer, select the remote computer name from your Partner List and connect. Once connected, see the remote desktop screen, take control of the mouse and keyboard, and work on it like you’re right in front of it. Put the remote computer to sleep and wake it up from within TeamViewer when you want to access it again.

    Backup and Restoration

    You may have questions regarding what backup and restoration solutions we provide. This article will help get answers.

    We back up all our shared hosting, and Hyper-V/VPS servers regularly, and that does not count against your disk quota. For dedicated servers, you can choose our Remote Data Center Backup (Windows Only) add-on service.

    VPS

    VHD backup

    1. We take the VHD backup every week. But the backup frequency depends on plan level. Usually, a VHD file will be available within 30-40 days.

  • Express and basic plan for Windows/Linux/Hyper-V VPS: Once per Four Weeks VHD Backup.
  • Professional and Advanced plan for Windows/Linux/Hyper-V VPS: Once per Two Weeks VHD Backup.

    • 2. Please note the VHD backup only contains data in SSD (C: partition). The whole SSD data will be restored and you cannot choose single file.

    3. It will cost $25 each time to restore from a VHD backup unless you order any one of the following add-on services: Remote Data Center Backup or Full Management Plan.

    4. The VHD backup file can be downloaded via FTP with $25/.vhd file. VHD restoration and VHD download are different services. Prices are calculated separately.

    5. Please open a ticket if you need us to restore your VPS or download the VHD file, and we will provide you with a list of available restore points to choose from.

    6. During a VHD restoration, your VPS will be offline for minutes. The period depends on your VHD size.

    Remote Data Center Backup

    We provide Remote Data Center Backup (Twice per week, every Wednesday and Saturday) with $14.99/month for a VPS. This service use Backup for Workgroups to back up files in drive C only to our backup server in remote data center. You can restore specified files/folders from backups at any time by yourself. Additionally, by ordering the add-on service, you can get free VHD restorations for 2 times each month.

    Shared Plan

    We backup shared plans twice every week.

    It will cost $25/request to restore the whole shared plan or certain files/website. Multiple files can be restored with one request, and we will just need a list of these files/directories. Subsequent requests will be treated as new requests.

    Dedicated Server

    We provide Remote Data Center Backup service for Windows dedicated servers only. If a dedicated server runs with non-Windows OS, such as CentOS, Ubuntu, Debian, etc, we suggest clients backup their own data.

    For Windows dedicated server, you can order Remote Data Center Backup service per your data size. This service will use Backup for Workgroups to backup your server data (C: partition only) to our remote data center servers twice per week. You can restore the backup files in your server at any time by yourself.
